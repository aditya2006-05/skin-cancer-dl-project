# skin-cancer-dl-project
🧠 Project Description

The Skin Cancer Detection Web App is a deep learning–powered application designed to assist in the early detection of skin cancer. Users can upload images of skin lesions, and the system analyzes them using a trained machine learning model to predict whether the lesion is benign or malignant.

The goal of this project is to provide a quick, accessible, and user-friendly tool that can support awareness and preliminary screening. It is not a replacement for professional medical diagnosis but serves as an assistive tool.

🚀 Features
📷 Upload skin lesion images for analysis
🤖 AI-based prediction using deep learning
📊 Displays prediction results with confidence score
🌐 Simple and responsive web interface
⚡ Fast inference and real-time results
☁️ Deployed online for easy access
